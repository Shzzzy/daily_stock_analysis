# 决策仪表盘

决策仪表盘是 AI 分析输出的结构化决策建议，包含买卖点位、止损设置和操作检查清单。

## 什么是决策仪表盘？

决策仪表盘是系统在完成股票分析后，AI 生成的结构化决策建议，包含明确的操作指导，帮助投资者做出决策。

**关键特征**:
- 一句话核心结论
- 精确买卖点位
- 止损/目标价格
- 操作检查清单（满足/注意/不满足）

## 数据结构

```python
@dataclass
class DecisionDashboard:
    stock_code: str              # 股票代码
    stock_name: str              # 股票名称
    decision: str                # 决策：买入/卖出/观望
    buy_price: Optional[float]   # 买入价
    stop_loss: Optional[float]   # 止损价
    target_price: Optional[float] # 目标价
    signals: List[str]          # 信号列表
    checklist: List[CheckItem]   # 检查清单
    summary: str                 # 一句话总结
```

### CheckItem

```python
@dataclass
class CheckItem:
    condition: str      # 条件描述
    status: str         # 满足/注意/不满足
    description: str    # 详细说明
```

{
  "decision": "买入",
  "buy_price": 1650.00,
  "stop_loss": 1600.00,
  "target_price": 1750.00,
  "summary": "MA5>MA10>MA20 多头排列，量比放大，可考虑买入",
  "signals": ["MA5>MA10>MA20", "量比放大", "突破前高"],
  "checklist": [
    {
      "condition": "趋势条件",
      "status": "满足",
      "description": "MA5 > MA10 > MA20，多头排列"
    },
    {
      "condition": "乖离率",
      "status": "注意",
      "description": "当前乖离率 4.5%，接近 5% 阈值"

    {
      "condition": "成交量",

## 内置交易纪律

| 规则 | 说明 |
|------|------|
| 严禁追高 | 乖离率超阈值（默认 5%）自动提示风险 |
| 趋势交易 | MA5 > MA10 > MA20 多头排列 |
| 精确点位 | 买入价、止损价、目标价 |
| 检查清单 | 每项条件以「满足/注意/不满足」标记 |
